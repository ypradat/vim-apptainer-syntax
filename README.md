# Vim Syntax Highlighting for Apptainer files

## Installation

### Makefile

```bash
git clone https://github.com/ypradat/vim-apptainer-syntax.git
cd vim-apptainer-syntax
make install
```

### Vundle

Add the following to your vimrc:

```vim
Plugin 'ypradat/vim-apptainer-syntax'
```
